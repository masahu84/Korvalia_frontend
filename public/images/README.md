# Imágenes del Hero Slider

Coloca aquí las imágenes para el slider del Hero:

- `hero1.jpg` - Primera imagen del slider
- `hero2.jpg` - Segunda imagen del slider
- `hero3.jpg` - Tercera imagen del slider

**Recomendaciones:**
- Tamaño: 1920x1080px mínimo
- Formato: JPG o WebP
- Peso: Optimizar para web (< 500KB por imagen)

**Nota:** Por ahora el slider usa imágenes de prueba. Cuando tengas las imágenes reales del diseño, simplemente reemplázalas en esta carpeta manteniendo los mismos nombres.
